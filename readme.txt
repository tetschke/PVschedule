Installation

In the following we state the system in which our computations were done. This code is provided as is. No support is provided, especially not for other system configurations. We used Ubuntu 18.04.4 LTS.

We used python 3.6.9 with the following packages:
casadi 3.4.5
matplotlib 3.1.0
numpy 1.16.0
sparse 0.8.0
